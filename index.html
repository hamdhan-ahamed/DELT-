TRANSISTOR-BASED LINE FOLLOWING ROBOT(Without Microcontroller)

This project aims to design and build a transistor based line following robot.

Description

This robot can follow a 2 cm wide black line on a white surface, detect obstacles in front, and stop, showing a red light. The main goal is to finish the robot using only individual components, without using microcontrollers, motor driver modules, or any programmable devices. The project outlines a complete low cost, transistor-based line-following robot design using discrete components only.

OutComes:
